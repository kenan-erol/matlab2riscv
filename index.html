CPSC 4900: Optimized Translation of MATLAB to RISC-V Architecture for Brain-Computer Interfaces: Final Report

Student: Kenan Erol

Advisor: Professor Rajit Manohar

Date: Spring 2026

Abstract

Brain computer interfaces (BCIs) offer significant clinical and therapeutic potential for neurological disorders such as epilepsy and stroke. Fully implantable BCIs impose severe limitations on power dissipation to prevent irreversible thermal damage to surrounding brain tissue. As a result, BCI hardware needs to balance real-time signal processing with ultra-low power consumption. The Hardware Architecture for LOw Power BCIs (HALO) project addresses these constraints through a design that combines application-specific hardware accelerators with a low-power RISC-V microcontroller equipped with the RISC-V Vector extension. This powerful design tradeoff enables software flexibility with hardware efficiency. However, there is a toolchain gap between the development languages preferred by neuroscientists and the low-level constraints of the HALO hardware that proves a hindrance to the widespread adoption of the technology. Algorithm development typically occurs in dynamically typed languages such as MATLAB, whereas HALO hardware requires statically typed, optimized, and vectorized C code. This report details the development of an end-to-end compilation pipeline engineered to bridge this divide. The pipeline uses a TypeScript-based compiler (ts-traversal) and an RVV-optimized C matrix library (RISCV-MATRIX) to translate MATLAB algorithms into power-efficient RISC-V binaries ready to run on HALO. Vectorized functions in RISCV-MATRIX have been shown to provide significant speedups, most strikingly being up to 95.43% faster for the newly vectorized mean test case on integer types. The report discusses the architectural design of the pipeline, benchmarks the full pipeline using the Spike instruction-set architecture simulator, investigates execution-cycle metrics, discusses translation benchmarks and fixes, and presents next steps for the research.
