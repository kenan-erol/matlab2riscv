% Port these seven traits from your humanities voice into this thesis:
%
%   1. OPEN WITH A HOOK. The EDST paper opens with Chris D'Agata's "relatively
%      civilized, relatively European..." quote. Do the same here: each
%      chapter should open with a vivid quote, stark image, or concrete
%      incident -- never with "The first phase was...".
%   2. STATE A THESIS EXPLICITLY. "I will argue that race and ethnicity are
%      the primary determinants of reception..." Your thesis here should be
%      similarly declarative, e.g. "I argue that compiler evolution -- not
%      hardware churn -- now drives the apparent performance ceiling of
%      hand-written RVV on HALO."
%   3. TOPIC SENTENCES DO REAL WORK. Each paragraph opens with a claim it
%      then defends. Replace narrative openers ("Further analysis was
%      conducted...") with the actual finding.
%   4. WEAVE EVIDENCE INTO PROSE. Integrate numbers and logs the way you
%      wove Krause/Haddad/Bleiker: quote, ellipsis, tight paraphrase, cite.
%      Your sources here are log files, commits, GCC release notes --
%      cite with the same discipline.
%   5. SYNTHESIZE, NOT SUMMARIZE. The essay groups Heck/Musk/Kassiri under
%      a single theme ("rigid baseline"). Cluster findings thematically,
%      not operation-by-operation.
%   6. CONCRETE ANCHOR -> ABSTRACT CLAIM. Every abstract claim trails a
%      specific example. Cf. every claim about state violence paired with
%      a named case (Sale v. Haitian Centers Council, Exodus v. Pence).
%   7. CLOSE WITH RESONANCE. The essay closes with rhetorical questions
%      and stakes ("what if this hierarchy were dismantled?"). Replace the
%      flat "Development for the remainder of the semester will..." with
%      a claim about what's at stake for BCI patients and for open-source
%      toolchains.
%
